# 效果图独立评审提示词模板

将完成的效果图作为附件交给 Mockup reviewer 子代理；页面级评审还要附上 `wireframe-spec.md`，全局方向评审还要附上产品简报。评审子代理只输出问题和方案，不修改效果图；控制代理负责向用户询问是否按方案修改。

```text
请以资深产品设计师的角度评审这张 App 界面。

重点检查：
1. 用户能否在 5 秒内理解页面目标
2. 视觉层级是否明确
3. 是否存在多个竞争性的主要操作
4. 信息密度是否合理
5. 字体、间距、圆角和对齐是否一致
6. 是否符合移动端可访问性要求
7. 哪些元素只是装饰，没有产品价值

请列出最重要的 5 个问题，并给出具体修改方案。
不要重新设计整个页面。
```
