# 全局视觉方向定位模板

在全局设计阶段使用本模板按需定位一至三套视觉方向。已有明确品牌或参考时只定义一套；用户要求探索或存在实质取舍时定义两至三套。此阶段只定义和冻结跨页面视觉语言，不生成页面、模块或代表页面效果图，也不创建 `.codex-workflow/visuals/global/` 图片。

冻结前将本模板保留在当前对话。用户明确选定并冻结方向后，只将选定方向、精简提示词哈希和冻结结果写入 `docs/design/global-design-freeze.md`。

## 全局方向冻结确认

一次只确认一个决策：

1. 选择哪套全局视觉方向。
2. 是否接受该方向定义的视觉签名和实现成本。
3. Global Freeze Signature Rule 适用时，签名是否可复述。
4. 是否使用 `pin`、`raise`、`loosen`，或不覆盖表达预算。
5. 是否明确冻结当前方向。

审计上述回答是否产生新的或冲突的范围、优先级、取舍、风险、验收或依赖。存在未决项时重新进入 `grilling`；不存在时直接冻结，不记录额外的“不适用”说明。不得把全局方向冻结当成任何页面效果图、页面设计冻结或实现批准。
